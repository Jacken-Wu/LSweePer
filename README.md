## 资源注释
* **以下图片资源可自行更改，注意尺寸**
* background(1200*640)如果像素不对将会强制转换
* block/flag/mine(40*40)
* title(250*100)
* size_text(220*30)
* BOOM(随便多大都行)
* WIN(随便多大都行)
* mask_layer(比background(1200*640)大就行，该图片只是起个遮罩作用，模糊背景的，防止背景太花哨看不清数字，过度影响扫雷)
